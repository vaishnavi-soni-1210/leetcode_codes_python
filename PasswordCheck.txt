Check if the password is valid.

Given an input string, which accepts only 8 characters from user. Check if it is a valid password.

Criteria:

1. there should be atleast 1 alphabet
2. there should be atleast 1 digit
3. special characters are allowed
4. space is not allowed
5. there should not be 3 consecutive alphabets or digits
6. minimum length of password should be 8 chars
